# SheShield-AI
🚨 AI-Powered Women Safety Application (SheShield AI)

An intelligent women safety system that uses AI, voice analysis, and real-time alerting to detect distress situations and automatically notify emergency contacts and authorities.

🌟 Features
🎤 Voice Distress Detection
Detects panic or distress from user voice using ML models
🧠 AI Emergency Summary Generation
Automatically generates a concise emergency report
📩 Instant Alerts (SMS/WhatsApp)
Sends alerts to emergency contacts using Twilio
📍 Live Location Tracking
Shares user’s real-time location in alerts
💬 NLP Emergency Chat Support
Users can type messages in critical situations
🔫 Weapon Detection (YOLO)
Detects dangerous objects via camera
